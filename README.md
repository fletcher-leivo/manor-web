# Blenheim Manor — Web Pages

Static web pages served by nginx for Blenheim Manor.

## Files

| File | Description |
|------|-------------|
| `index.html` | Manor landing page — services and navigation |
| `household.html` | Household org chart — staff roles and structure |

## Deployment

These files are served from `/var/www/html/` via nginx. This repo exists for version control and collaboration — the live copy is synced manually.
